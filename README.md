## 🚀 How to Run

**Step 1 — Install libraries:**
```bash
pip install -r requirements.txt
```

**Step 2 — Run the script:**
```bash
python dashboard.py
```

**Output:**
- Chart will open on screen automatically
- Image saved as `dashboard_output.png` in same folder
